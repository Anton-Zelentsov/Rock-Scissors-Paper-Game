# ✊✌️✋ Камень, Ножницы, Бумага

Простая desktop-игра с графическим интерфейсом на Python и **Tkinter**. Нажимаешь любую кнопку — программа случайно выбирает один из трёх вариантов и показывает результат на экране.

## 🚀 Запуск

```bash
python game.py
```

Tkinter входит в стандартную библиотеку Python, дополнительно ничего ставить не нужно.

## 🔑 Ключевые технические решения

### 1. GUI на Tkinter без внешних зависимостей
Весь интерфейс собран из стандартных виджетов: окно `Tk()`, текстовая метка `Label` и три кнопки `Button`. Не требуется ни одной сторонней библиотеки.

### 2. Фиксированный размер и позиционирование через `place`
Окно сделано нерастягиваемым, а элементы расставлены по точным координатам:

```python
root.resizable(width=False, height=False)
rock.place(x=50, y=300)
```

Метод `.place(x, y)` даёт полный контроль над расположением — кнопки всегда стоят там, где задумано, независимо от содержимого.

### 3. Случайный выбор через `random.choice`
Логика игры — одна функция: из списка вариантов случайно берётся один и подставляется в текст метки.

```python
def rock_scissors_paper():
    rsp = ['Камень', 'Ножницы', 'Бумага']
    value = choice(rsp)
    lable_text.configure(text=value)
```

### 4. Одна функция-обработчик на все кнопки
Все три кнопки вызывают один и тот же `command`. Это убирает дублирование: вместо трёх отдельных обработчиков — единая точка логики.

### 5. Обновление интерфейса в реальном времени
Результат не печатается в консоль, а меняется прямо в окне через `configure(text=...)` — метка перерисовывается при каждом нажатии.

## 🛠️ Технологии

- **Python 3**
- **Tkinter** — стандартная библиотека для GUI
- **random** — генерация случайного выбора
